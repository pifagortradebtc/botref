# Bybit Referral Checker

Telegram Mini App для проверки рефералов Bybit по UID.

## Установка

```bash
git clone https://github.com/pifagortradebtc/botref.git
cd botref
npm install
cp .env.example .env
```

Заполните `.env`: `BYBIT_API_KEY`, `BYBIT_API_SECRET` (разрешение Affiliate).

## Запуск

```bash
npm start
```
